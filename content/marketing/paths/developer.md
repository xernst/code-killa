## who this is for
you write code, or you're trying to. you use cursor or claude code every day,
and you can feel the gap between "it ran" and "i know why it ran." this path
closes that gap.

## what you'll be able to do
read, debug, and direct any ai-generated python on sight. understand mutation,
state, errors, and the agent loop well enough to catch what the model got
wrong. ship and maintain a production agent harness. you finish by building a
working cli agent in the capstone.

## the path
the full course, 35 chapters: the python foundations, real python, the command
line and the ai clis, llm apis, shipping discipline, the capstone, and the
applied builds. about 20 to 24 hours, self-paced. nothing is gated.

## how it works
no videos, no fizzbuzz. every step shows a small piece of ai-written code. you
read it, predict what it does, find the bug, fix it, run it. it all runs in
your browser.
