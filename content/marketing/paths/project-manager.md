## who this is for
you manage products or projects. your team ships ai features, you use cursor
to draft specs, and every time you try to actually change the generated code
you get stuck. you want to direct the work, not just route it. this path gets
you there.

## what you'll be able to do
read ai-generated python well enough to review it and sign off honestly. brief
an ai tool the way you'd brief a good contractor. read an agent trace to see
what actually happened. run an eval to know if a feature is good enough to
ship. and build a small internal tool without waiting on engineering.

## the path
fourteen chapters: the python foundations, the terminal, calling ai models,
structured output, prompting, agent traces, evals, production tradeoffs,
debugging output, and team skills. about 12 to 14 hours, self-paced.

## how it works
no videos, no fizzbuzz. every step shows a small piece of ai-written code. you
read it, predict what it does, find the bug, fix it, run it. it all runs in
your browser.
