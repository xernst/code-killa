## who this is for
you run operations. you've watched ops roles get replaced by automations, and
you'd rather be the person who builds them. you use ai tools but can't yet
build the workflow that runs without you. this path gets you there.

## what you'll be able to do
build automations that loop through real work and handle the failures instead
of crashing. pull data, call ai models, and wire them into the processes your
department depends on. build and maintain an agent harness. understand the
terminal and the ai clis well enough to actually ship.

## the path
eighteen chapters: the python foundations, the terminal, the claude cli,
modules, error handling, files and i/o, http and apis, calling ai models,
structured output, mcp, agent loops, and agent harnesses. about 16 to 18 hours,
self-paced. the second-most technical path after developer.

## how it works
no videos, no fizzbuzz. every step shows a small piece of ai-written code. you
read it, predict what it does, find the bug, fix it, run it. it all runs in
your browser.
